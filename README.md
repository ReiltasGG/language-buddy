# language-buddy 🌍

A locally-run AI language learning web app powered by **Gemma 3** and **Flask**. Practice conversation and vocabulary in 8 languages with progress tracking — all running on your machine with no subscriptions or API costs.

---

## Features

### 💬 Conversation Mode
- Chat with an AI tutor in your target language
- Responses include a **📝 Notes** section with:
  - English translation of what the tutor said
  - Gentle corrections of any mistakes you made
  - Grammar and vocabulary tips
- Conversation history kept for context
- Encouraging and beginner friendly tone

### 🃏 Flashcard Mode
- Dynamic flashcards generated by Gemma 3 for your selected language
- 4 categories for Japanese, Korean, and Mandarin / 3 for all other languages:
  - **Verbs** — common action words
  - **Places & Names** — locations and proper nouns
  - **Common Nouns** — everyday vocabulary
  - **Kanji** (Japanese) / **Hangul** (Korean) / **Hanzi** (Mandarin) — script practice
- Furigana / romanization / pinyin shown under characters where applicable
- Flip animation to reveal the answer
- Correct / Incorrect tracking with session score and progress bar

### 🔁 Review Mode
- Automatically pulls the cards you've gotten wrong most
- Same flip card UI as flashcard mode
- Updates your progress as you review
- "Review Again" restarts the session without leaving the page

### 📊 Stats Page
- Tracks all your progress in a local SQLite database
- Per language stats:
  - Total cards studied
  - Overall accuracy
  - Breakdown by category
  - Cards you struggle with most
- Reset button to wipe progress for a specific language and start fresh

## Setup

**Requirements:**
- Python 3
- [Ollama](https://ollama.com) with `gemma3:4b` installed

**1. Clone the repo**
```bash
git clone https://github.com/ReiltasGG/language-buddy.git
cd language-buddy
```

**2. Create and activate a virtual environment**
```bash
python3 -m venv venv
source venv/bin/activate
```

**3. Install dependencies**
```bash
pip install flask requests
```

**4. Pull the model**
```bash
ollama pull gemma3:4b
```

**5. Start Ollama in a separate terminal**
```bash
ollama serve
```

**6. Run the app**
```bash
python app.py
```

**7. Open in your browser**
```
http://127.0.0.1:5001
```

> **Note:** Port 5000 is used by AirPlay Receiver on Mac. If you get a port conflict, go to System Settings → AirPlay Receiver and turn it off, or change the port in `app.py`.

## Notes

- Flashcards are dynamically generated each session by Gemma 3
- Progress is saved locally in `progress.db` and persists between sessions
- Response speed depends on your hardware — an Apple M1 Pro runs `gemma3:4b` comfortably
- No internet required once the model is downloaded
